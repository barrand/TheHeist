# Examples Directory

This directory contains **documentation examples and design references only**.

These files are NOT loaded by the game - they serve as:
- Design documentation
- Format examples  
- Planning references
- Dependency tree visualizations

## Contents

- `museum_gala_dependency_tree.md` - Visual task dependency flow for museum scenario
- `train_robbery_dependency_tree.md` - Visual task dependency flow for train scenario

## Note

**Playable experience files** belong in `/backend/experiences/`, not here.

See `/backend/experiences/README.md` for details on playable experience organization.
